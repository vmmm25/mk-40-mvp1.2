## Step 3: Write the Post

### Format Rules
- **Max 1.300 characters** for the main body
- **Line breaks** between every paragraph (mobile readability)
- **One idea only** — Don't try to cover multiple topics
- **Hook in first 2 lines** — This is what shows before "ver más"
- **Max 5 hashtags** at the end
- **No link in main body** — Links kill organic reach. Note: "Link in first comment" at the end

### Structure by Type

**Thought Leadership:**
```
[Hook: Contrarian take, surprising stat, or provocative question]

[2-3 short paragraphs with line breaks]

[Practical insight from experience]

[Soft CTA: question to audience or invitation to discuss]

#ISO27001 #Ciberseguridad #Compliance #RGPD #Startups
```

**Client Win:**
```
[Opening: The challenge — anonymized if needed]

[What we did — specific]

[Result — metric or tangible outcome]

[Takeaway for the reader]

[Soft CTA]
```

**Regulatory Update:**
```
[What changed — plain language]

[Who's affected]

[What to do — 2-3 actionable points]

[Note: full analysis link in first comment]
```

**Content Promotion:**
```
[Hook: The problem the blog post solves]

[Key insight from the post]

[Why it matters now]

[CTA: "Te cuento más en el artículo (link en primer comentario)"]
```
