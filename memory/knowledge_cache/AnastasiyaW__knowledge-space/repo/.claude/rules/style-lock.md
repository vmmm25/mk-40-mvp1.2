# Style Lock - согласованные стили

## Stats bar (.ks-graph-stats)
Лейблы должны быть КОРОТКИМИ (одно слово): articles, links, communities, domains.
НЕ использовать: cross-references, topic communities, и другие многословные.
font-size лейблов: 0.58rem, nowrap обязателен.

## Snippet (.claude-snippet)
Кнопка Copy в dark theme: белая (rgba(255,255,255,0.12) bg, rgba(255,255,255,0.85) text).
НЕ фиолетовая (#bb86fc).

## Contributors (.ks-contributors)
LinkedIn иконка в .ks-contributor__avatar (синий фон #0a66c2, белая иконка).
НЕ голый SVG напрямую.

## Header
Две кнопки GitHub в ряд (knowledge-space + claude-code-config).
Логотип = animated favicon SVG, не дефолтная иконка книги.

## Общее
- `---` (hr) разделители: максимум 1 на странице (перед footer). НЕ между каждой секцией.
- hr margin: 1.2rem, не 2rem+.
