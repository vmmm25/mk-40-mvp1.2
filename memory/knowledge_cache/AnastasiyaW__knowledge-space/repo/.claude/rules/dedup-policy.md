# Deduplication Policy - same topic name does NOT mean skip

When processing new content (books, courses, research) and an article on the same topic already exists:

1. **Read the existing article first**
2. **Compare** with new content:
   - If new data **contradicts** existing -> do deep research, update article with verified info, explain WHY our version is correct
   - If new data **adds unique value** (alternative approaches, new examples, edge cases, gotchas) -> enrich the article
   - If new data is an **exact copy** of what we already have -> skip
3. **Never skip just because the topic name matches** - different sources provide different perspectives

When sources disagree:
- Research the topic independently
- Add a note in the article explaining the discrepancy and why our conclusion is what it is
- Example: "Some sources suggest X, but since version Y this was changed to Z (see official docs)"
