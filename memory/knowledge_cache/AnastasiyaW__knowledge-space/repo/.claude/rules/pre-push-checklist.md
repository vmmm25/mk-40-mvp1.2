# Pre-Push Stats Checklist

After adding articles or domains, BEFORE pushing, run this check:

```bash
# Get actual counts
ARTICLES=$(find docs/ -name "*.md" -not -name "index.md" -not -path "*/blog/*" -not -path "*/contributing/*" -not -name "for-llm-agents.md" -not -name "privacy.md" | wc -l)
DOMAINS=$(ls -d docs/*/ | grep -v blog | grep -v contributing | grep -v javascripts | grep -v stylesheets | grep -v assets | grep -v knowledge-base | wc -l)
echo "Actual: $ARTICLES articles, $DOMAINS domains"

# Check all hardcoded locations
echo "=== Stale values ==="
grep -rn "560\|608\|620\|621\|622\|629\|650\|667" \
  README.md AGENTS.md docs/index.md docs/blog/posts/welcome.md \
  mkdocs.yml .claude/rules/article-rules.md 2>/dev/null \
  | grep -iv "viewBox\|path\|svg\|base64\|charset\|node_id"

grep -rn " 22 domain\| 23 domain\| 24 domain\| 25 domain\| 26 domain" \
  README.md AGENTS.md docs/index.md docs/blog/posts/welcome.md \
  mkdocs.yml CONTRIBUTING.md .claude/rules/article-rules.md 2>/dev/null
```

## Files to update (manual):

| File | What | grep pattern |
|------|------|-------------|
| `README.md` | "XXX+ articles \| YY domains" | articles.*domains |
| `README.md` | "across YY domains" | across.*domains |
| `AGENTS.md` | "XXX+ articles, YY domains" | articles.*domains |
| `docs/index.md` | "across YY domains" (3 places) | across.*domains |
| `docs/index.md` | snippet "XXX+ articles across YY" | articles across |
| `docs/blog/posts/welcome.md` | "XXX+ dense reference" | dense reference |
| `docs/blog/posts/welcome.md` | "across YY technical" | across.*technical |
| `mkdocs.yml` | site_description "XXX+ curated" | curated articles |
| `mkdocs.yml` | "and NN more domains" | more domains |
| `.claude/rules/article-rules.md` | "XXX+ articles across YY" | articles across |
| GitHub description | `gh repo edit --description` | - |

## Files auto-updated at build time (no action needed):
