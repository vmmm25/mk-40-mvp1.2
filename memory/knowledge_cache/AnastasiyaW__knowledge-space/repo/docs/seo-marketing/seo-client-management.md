# SEO Client Management and Business

Client communication protocols, pricing models, work scope definition, project management, legal documents, and career development for SEO specialists.

## Soft Skills vs Hard Skills

Hard skills determine grade level. Soft skills determine client retention and revenue ceiling. A middle specialist with strong soft skills outperforms a senior who ignores communication.

Client lifetime is directly tied to soft skill quality, not technical depth alone.

## Work Scope Lists

### Two-Part Structure: Process + Result
Every work item needs:
- **Process**: what actions are performed
- **Result**: what deliverable is produced

Example for semantic core: Process = competitor analysis -> marker collection -> Wordstat parsing -> frequency -> cleaning -> clustering -> refinement. Result = clustered keyword table with frequencies.

### 6-Month Work Plan Template (Mid-Size E-Commerce)
- Month 1: Technical audit + competitor analysis + commercial factors audit
- Month 2-4: Semantic core + text optimization (H1/URL/title/description + briefs)
- Month 3-5: Link mass development (start month 3-4)
- Ongoing: Analytics, position tracking, iterative optimization

## Capacity and Pricing

### Project Capacity
- Standard: 4-6 projects per specialist
- Optimal: 4 projects
- Calculation: 160 hours/month / 4 = 40 hours per project

### Time Budget Examples
- Semantic core (medium): 15-20 hours
- Text optimization: 7-10 hours
- Monthly analytics/reporting: 3-5 hours
- Technical audit: 10-20 hours (first month)
- Link audit: 5-8 hours

### Agency Format Types

| Format | Projects/Specialist | Quality | Price Range |
|--------|---------------------|---------|-------------|
| Conveyor | 20-40 | Low | 7-30k RUB/month |
| Mid-tier | 6-12 | Medium | 30-80k RUB/month |
| Boutique | 3-6 | High | 80-300k+ RUB/month |

### Pricing Models
1. **Fixed monthly** - recommended for beginners
2. **Fixed + KPI bonus** - reduce fixed by 30-50%, add traffic/leads bonus
3. **Revenue share** - experienced specialists only

### Hourly Rate System (Recommended)
```yaml
Monthly fee = Hours x Hourly rate
Example: 40 hours x 750 RUB = 30,000 RUB/month
```

**Minimum floors**: 500 RUB/hour; 15,000 RUB/month minimum (preferably 20,000+).

### Additional Client Expenses (Discuss Upfront)
- Development: 10-15 hours/month (first 2-3 months)
- Copywriting: 10-15k RUB/month
- Links: 10-15k RUB/month for competitive niches
- Tools: your cost, not client's

## Communication Protocols

### Meeting Frequency
- Minimum: monthly
- Recommended: bi-weekly
- Premium: weekly

### Meeting Notes
Document all agreements immediately: new tasks, changes, feedback, commitments. Write into project management tool - never rely on memory.

### Communication Principles
- Translate SEO into business impact (traffic -> leads -> sales -> revenue)
- Report issues before client notices (proactivity)
- Over-deliver on small extras (builds disproportionate loyalty)
- Respond to emotional clients with calm rationality

### Conflict Resolution
1. Lower emotional temperature
2. Acknowledge concern as valid
3. Explain with facts and data
4. Propose concrete resolution options
5. Add small bonus if appropriate
6. Document in writing

## Project Management

### Recommended Tools
- Trello (beginners, free, simple kanban)
- Notion (flexible, combines tasks + docs)
- Asana (cleaner UI)

### Board Structure
Columns: To Do -> In Progress -> In Review -> Done -> Control -> Archive

### Task Card Format
Name: **Verb + Object** ("Correct slash redirect" not "redirect")
Description: Concept (for client) -> Problem (what's wrong) -> Solution (for developer) -> Example (specific URL)

## Legal Documents

### Contract Requirements
- Specific domain being promoted
- Analytics tool definitions
- 30-day billing cycle
- Detailed service scope
- 100% prepayment
- Service acceptance timeline
- Client obligations: analytics access, implement recommendations
- Email as legally binding communication
- **No guarantees clause** (rankings cannot be guaranteed)

### Document Flow
1. Contract (before work)
2. NDA if required
3. Payment invoice (each cycle)
4. Service delivery act (after each period)

## Finding First Clients

### Channel 1: Personal Network
Contact friends, family, acquaintances. Ask for referrals two levels deep.

### Channel 2: Partner Referral Network
Find adjacent specialists (developers, PPC, copywriters) on Avito, FL.ru, Profi.ru. Offer mutual referrals with 10-20% monthly commission. Build pipeline of 50-100 potential partners.

### Channel 3: Offline Networking
Local business meetups. Slower but higher conversion.

## Client Lifetime Extension

1. Deliver all tasks on time, every time
2. Fix minor extras without charging
3. Document everything
4. Report proactively
5. Build personal rapport
6. Handle crises with solutions, not excuses

The difference between 3-month and 3-year client is almost entirely soft skills.

## Portfolio and Cases

### Portfolio Structure
Per project: type, starting state, work performed, results with evidence, timeframe. Minimum 3-4 projects.

### Case Study Structure
1. Problem statement
2. Actions taken
3. Results (metrics before/after)
4. Evidence (screenshots)
5. Timeline
6. Client testimonial (if available)

Write cases even when not looking for work - compounds value over time.

## Gotchas
- **Never guarantee rankings in contracts** - rankings depend on external factors beyond your control
- **15k RUB minimum floor is critical** - below that creates psychological anchoring that devalues future work
- **Meeting notes in project tool, not memory** - prevents all "but we agreed..." disputes
- **Translate SEO into money, not metrics** - clients care about revenue, not position changes
- **Conveyor agencies (20-40 projects) produce low quality** - filter when job hunting
- **First 2 weeks at new job: observe, don't change** - understand existing systems before suggesting improvements
